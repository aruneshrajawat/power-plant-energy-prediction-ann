# AI Energy Intelligence Dashboard

Deep Learning powered ANN system to predict power plant energy output using environmental parameters.

## Technologies Used
• PyTorch
• Streamlit
• Pandas
• Plotly

## Features
• AI prediction dashboard
• Real-time energy prediction
• Data visualization
• Correlation analysis
• Interactive sliders
• Scatter analysis (Temperature vs Power Output)

## Dataset
The model uses environmental parameters to predict power plant energy output:
• Temperature (AT)
• Vacuum (V)
• Pressure (AP)
• Humidity (RH)

## Model Architecture
Artificial Neural Network (ANN)

Input Layer → 4 Features  
Hidden Layer → 6 Neurons  
Hidden Layer → 6 Neurons  
Output Layer → Power Output Prediction
